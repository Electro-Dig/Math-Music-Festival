# Math & Music Festival Collection

This repository is a collective showcase of various interactive web projects that explore the fascinating intersections between Artificial Intelligence, Mathematics, and Music. Created to experiment with geometric patterns, physics simulations, audio synthesis, and interactive computer vision tools.

## Included Projects

1. **毕达哥拉斯分形树 (Pythagoras Fractal Tree)**
   A geometric fractal tree generator that produces beautiful visualizations through iterative scaling and rotation.

2. **函数舞蹈 (Function Dance)**
   A creative coding project exploring the choreographic patterns formed by mathematical equations and trigonometric functions.

3. **克拉德尼板模拟器 (Chladni Plate Simulator)**
   Interactive simulation of Chladni figures, demonstrating standing wave patterns in vibrating plates using classical physics.

4. **Hand & Music 2x2**
   An AI-powered music generation tool using hand gestures to control a matrix of sound elements, blending computer vision with audio art.

5. **Camera Kaleidoscope**
   Transforms standard webcam feeds into mesmerizing interactive kaleidoscope patterns in real-time.

6. **Hands Whitney**
   An audiovisual experience inspired by John Whitney, driven by real-time hand-tracking technology to compose visual music.

## How to Run

Most projects are built using static HTML/JS/CSS or modern web frameworks like Vite/React.
- **For static HTML projects**: Simply open the respective `index.html` file in your preferred web browser.
- **For Vite/React projects**:
  1. Navigate to the project folder (e.g., `cd 毕达哥拉斯分形树`).
  2. Install dependencies: `npm install`.
  3. Start the dev server: `npm run dev`.

An aggregate `index.html` portal is provided in the root directory to easily link to all these projects (especially helpful if deploying this repository to a static hosting service like GitHub Pages).
